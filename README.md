# 🛒 Simple E-commerce Store

This is a basic full-stack e-commerce web application built as part of an internship task. It includes user authentication, product listings, shopping cart functionality, and order placement.

## 💻 Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)


## 🚀 Features

- User registration & login (JWT-based)
- Product listing & detail view
- Shopping cart (localStorage)
- Order placement (protected API)
